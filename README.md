# Puppeteer

## Cài đặt

```
    npm i puppeteer
    # or yarn add puppeteer
```

## Note
#### Tìm kiếm thư mục của Chrome
Truy cập browser chorme, gõ 'chrome://version/'

Tìm nội dung 'Executable Path'

Vd: Executable Path: /Applications/Google Chrome.app/Contents/MacOS/Google Chrome
